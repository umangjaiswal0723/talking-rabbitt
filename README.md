# 🐇 Talking Rabbitt

An AI-powered Business Intelligence and Data Analytics Dashboard.

## 🌐 Live Demo
[https://talking-rabbitt-4lvq.onrender.com](https://talking-rabbitt-4lvq.onrender.com)

## 📊 Features
- Upload CSV and Excel files
- Auto-generated KPI dashboard
- Interactive charts and visualizations
- AI-powered chat to ask questions about your data
- Trend detection and anomaly detection
- Forecasting / predictions
- PDF export

## 🛠️ Tech Stack
**Frontend**
- React + Vite
- Tailwind CSS
- Recharts (charts)
- Axios (API calls)

**Backend**
- FastAPI (Python)
- Pandas (data analysis)
- Scikit-learn (forecasting)

**AI**
- Groq API (llama-3.3-70b-versatile)

## 🚀 Run Locally

**Backend**
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000

**Frontend**
cd frontend
npm install
npm run dev
